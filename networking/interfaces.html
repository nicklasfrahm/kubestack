    Interfaces¶

    The Interface custom resource allows you to manage network interfaces on your network devices. All interfaces must reference a Connection. For more information about how to configure a Connection, please refer to the management section.

    You may get a list of network interfaces by running the following command.

    kubectl get if

    NAME                  CONNECTION-NAME   SELECTOR-NAME   VLAN-MODE
distswitch00-eth1-1   distswitch00      eth1/1          Access
distswitch00-eth1-3   distswitch00      eth1/3          Trunk

    VLAN¶

    Virtual local area networks (VLANs) are used to segment a network into multiple broadcast domains. They allow you to build multiple networks using a single physical network. For more information about VLANs, please refer to the VLAN Wikipedia article.

    Access port¶

    An access port is a link that is configured to only allow traffic for a single VLAN. This is useful to connect an end-user device to a specific VLAN.

    distswitch00-eth1-1.yaml
    apiVersion: kubestack.nicklasfrahm.dev/v1alpha1
kind: Interface
metadata:
  # (optional) These labels are not required for Kubestack to work.
  labels:
    app.kubernetes.io/name: interface
    app.kubernetes.io/instance: distswitch00-eth1-1
    app.kubernetes.io/part-of: kubestack
    app.kubernetes.io/managed-by: kustomize
    app.kubernetes.io/created-by: kubestack
  name: distswitch00-eth1-1
spec:
  # (required) Configure the connection.
  connectionRef:
    # (required) The name of the Connection resource.
    name: distswitch00
    # (optional) Specify the namespace. Defaults
    # to the namespace of the Connection resource.
    # namespace: default
  # (required) Specify an interface selector.
  selector:
    # (optional) The name of the interface.
    name: eth1/1
    # (optional) The MAC address of the interface.
    # mac: 00:00:00:00:00:00
  # (optional) Override the interface name.
  # This may not be supported by all appliances.
  # name: eth1x01
  # (required) Specify the interface type.
  type: EthernetCSMACD
  # (optional) Specify the interface MTU.
  # mtu: 1500
  # (optional) Specify the interface description.
  description: Internet towards ISP.
  # (optional) Enable or disable the interface.
  enabled: true
  # (optional) Specify additional ethernet configuration.
  ethernet:
    # (optional) Override the system-assigned MAC address.
    # macAddress: 00:00:00:00:00:00
    # (optional) Enable or disable auto-negotiation.
    autoNegotiate: true
    # (optional) Force the duplex mode.
    # duplexMode: Full
    # (optional) Force the speed.
    # portSpeed: 1Gbps
    # (optional) Configure switched VLANs.
    switchedVLAN:
      # (required) Configure the VLAN mode.
      interfaceMode: Access
      # (required) Configure the access VLAN ID.
      accessVLAN: 5

    Trunk port¶

    A trunk port is a link that is configured to allow traffic for multiple VLANs. This is useful to connect two VLAN-aware switches together.

    distswitch00-eth1-3.yaml
    apiVersion: kubestack.nicklasfrahm.dev/v1alpha1
kind: Interface
metadata:
  # (optional) These labels are not required for Kubestack to work.
  labels:
    app.kubernetes.io/name: interface
    app.kubernetes.io/instance: distswitch00-eth1-3
    app.kubernetes.io/part-of: kubestack
    app.kubernetes.io/managed-by: kustomize
    app.kubernetes.io/created-by: kubestack
  name: distswitch00-eth1-3
spec:
  # (required) Configure the connection.
  connectionRef:
    # (required) The name of the Connection resource.
    name: distswitch00
    # (optional) Specify the namespace. Defaults
    # to the namespace of the Connection resource.
    # namespace: default
  # (required) Specify an interface selector.
  selector:
    # (optional) The name of the interface.
    name: eth1/3
    # (optional) The MAC address of the interface.
    # mac: 00:00:00:00:00:00
  # (optional) Override the interface name.
  # This may not be supported by all appliances.
  # name: eth1x03
  # (required) Specify the interface type.
  type: EthernetCSMACD
  # (optional) Specify the interface MTU.
  # mtu: 1500
  # (optional) Specify the interface description.
  description: Internet towards ISP.
  # (optional) Enable or disable the interface.
  enabled: true
  # (optional) Specify additional ethernet configuration.
  ethernet:
    # (optional) Override the system-assigned MAC address.
    # macAddress: 00:00:00:00:00:00
    # (optional) Enable or disable auto-negotiation.
    autoNegotiate: true
    # (optional) Force the duplex mode.
    # duplexMode: Full
    # (optional) Force the speed.
    # portSpeed: 1Gbps
    # (optional) Configure switched VLANs.
    switchedVLAN:
      # (required) Configure the VLAN mode.
      interfaceMode: Trunk
      # (optional) Configure the VLAN ID of untagged
      # frames arriving at this port. Defaults to 1.
      nativeVLAN: 9
      # (required) Configure the trunk VLAN IDs.
      trunkVLANs:
      - 5
      - 9
      - 10
